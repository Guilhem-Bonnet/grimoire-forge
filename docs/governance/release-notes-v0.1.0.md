# Release Notes v0.1.0

## Titre

Grimoire Forge v0.1.0 - Baseline publique du moteur de creation de projets agentiques.

## Resume

Cette release etablit la premiere baseline publique de Grimoire Forge.
Elle formalise la vision produit, la roadmap v1, le backlog initial et le cadre de contribution open source.

## Highlights

- Repositionnement explicite du projet vers l'objectif moteur agentique.
- Documentation structuree en trois axes : vision, roadmap, gouvernance.
- Roadmap publique v1 et backlog initial priorise.
- Templates de contribution GitHub pour issues et pull requests.
- Message de lancement public pret a diffuser.
