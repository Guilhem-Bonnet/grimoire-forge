# Annonce - Lancement Public de Grimoire Forge

## Message court

Grimoire Forge est maintenant ouvert publiquement.

Le projet vise un objectif clair: construire un moteur de creation de projets agentiques, avec generation structuree d'artefacts, qualite integree et gouvernance open source actionnable.

## Message detaille

Nous ouvrons Grimoire Forge pour accelerer la construction d'un moteur de creation de projets agentiques fiable et reutilisable.

Ce que vous trouverez des maintenant:

- Une vision produit explicite.
- Une roadmap publique v1 orientee execution.
- Un backlog initial priorise.
- Des templates d'issues et de pull requests pour contribuer rapidement.

Ce que nous recherchons:

- Retours terrain sur les cas d'usage prioritaires.
- Contributions sur le moteur de generation et la qualite.
- Ameliorations de documentation pour reduire la friction d'onboarding.
