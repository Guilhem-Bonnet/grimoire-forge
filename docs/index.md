# Grimoire Forge - Documentation

## Vue d'ensemble

Cette documentation définit la cible du projet : construire un moteur de creation de projets agentiques, utilisable en environnement reel, avec un socle de qualite et de gouvernance.

## Principes

- Les decisions sont traçables et versionnees.
- Les workflows restent executables par des agents et des humains.
- Chaque evolution passe par validation qualite avant diffusion.
