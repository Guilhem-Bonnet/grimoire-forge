!!! info "Pourquoi ce projet"
    Grimoire Forge assemble orchestration BMAD, architecture runtime, controle qualite et standards documentaires pour industrialiser la creation de produits agentiques sans perdre la vitesse d'iteration.

## Capacites strategiques { .section-heading }

- **Orchestration BMAD native**  
  Agents specialises, handoffs explicites et workflows pilotables dans une meme chaine de valeur.
- **Qualite operationnelle continue**  
  Verifications automatiques, gates de preflight et traces de decision pour limiter les regressions.
- **Documentation executable**  
  Les artefacts restent lisibles pour les humains et exploitables par des agents sans traduction manuelle.
- **Gouvernance open source**  
  Conventions, checklists de release et publication transparente pour une execution reproductible.
