# Roadmap Publique v1 - Grimoire Forge

## Ambition v1

Livrer un moteur de creation de projets agentiques fiable, orienté execution, avec une gouvernance qualite native.

## Piliers v1

1. Generation agentique structuree.
2. Qualite obligatoire et verifiable.
3. Documentation executable pour contributeurs externes.
4. Gouvernance open source claire.

## Chantiers prioritaires

### Chantier A - Moteur de generation

- Flux standard intention -> specification -> artefacts -> validation.
- Contrats d'entree/sortie des generateurs formalises.
- Verification de coherence inter-artefacts.

### Chantier B - Qualite

- Pipeline commun lint, tests, preflight, memory-lint.
- Non-regression sur les generateurs critiques.
- Rapport de sante projet a chaque cycle majeur.

### Chantier C - Experience contributeur

- Templates issue/PR operationnels.
- Parcours de contribution documente.
- Exemples reproductibles de bout en bout.

### Chantier D - Publication et gouvernance

- Hygiene open source complete.
- Backlog public priorise.
- Rituels de triage et de review explicites.

## Jalons de validation

- Jalon 1: socle de generation stable.
- Jalon 2: pipeline qualite bloqueur operationnel.
- Jalon 3: onboarding contributeur fluide.
- Jalon 4: publication publique et cycle de contribution actif.

## Indicateurs de reussite

- Un projet agentique peut etre initialise de facon reproductible.
- Les regressions critiques sont detectees avant integration.
- Un contributeur externe peut proposer une PR sans accompagnement synchrone.
- Le backlog public reste maintenu et priorise.
